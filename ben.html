A quick guide to HTML tags


There are about 10 to 15 crucial tags. Tags are shown by a less than sign, some letter and/or number and a greater than sign. For example <p> is a paragraph tag. It's what I'm using right now. This kind of tag requires a closing tag, so after we're done writing we can put a </p>to "close" the element (tag)

The first sentence you read on this site is large! That's a header tag. Headers range from <h1> to <h6>. These should probably come at the top of your page, or when you want to get an idea across.


Table of HTML elements


  1. Lists are formed with a <ul> with <li> elements nested in it.
  2. Links are created with <a href="somepage.html">Click me!</a>
  3. Images are inserted by writing <img src="photoname.jpg"/>
  4. You can add line breaks with <br>
  5. All HTML files must start and be nested with an <html> tag!
  6. After your HTML tag, you need a <head> tag which the browser will use and then close it
  7. Last you need a <body> tag. This is where all your content will go!
  8. Also, buttons! They're very helpful. <button> and </b> will give you a button!
